# **App Name**: Astrotype

## Core Features:

- Theme Design: Display blog posts in a clean, Zaggonaut-inspired theme.
- Static Page Generation: Generate static blog pages using Astro.
- Deployment Automation: Include GitHub Actions workflow for continuous deployment to GitHub Pages on new blog posts.
- Domain Integration: Allow for hosting under a personal domain with DNS configuration instructions provided.
- GitHub Repo Setup: Include setup guide for connecting site to github
- Category Suggestions: Provide suggestions for post categories or tags, powered by a generative AI tool trained on the post contents and metadata.

## Style Guidelines:

- Primary color: black (#141619) for a calming and modern aesthetic.
- Background color: Off-white (#F5F5F5) to ensure readability and a clean feel.
- Accent color: Soft green (#0016EC) to highlight important elements.
- Body and headline font: 'Exo 2' (sans-serif) for a modern and readable style. Note: currently only Google Fonts are supported.
- Minimalist icons for categories and navigation. Consider using Feather or Tabler icons.
- Clean and spacious layout, similar to Zaggonaut, with a focus on readability.
- Subtle transitions for page loading and interactions to enhance user experience without being distracting.
